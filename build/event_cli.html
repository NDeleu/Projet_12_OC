Create an event

Parameters:

name (str): Event Name in alphabetical value.

event_start (datetime): Event start date in this form: YYYY/MM/DD-HH:MM:SS.

event_end (datetime): Event end date in this form: YYYY/MM/DD-HH:MM:SS.

location (str): Event location in alphabetical value.

attendees (int): Attendees amount in numerical value.

instruction (str): Some instructions for the event in alphabetical value.

contract (int): Contract Id in numerical value.
