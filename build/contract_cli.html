main.py contract read¶

Get a list of contracts

Options:

–mine (bool): If you are seller, filter awarded contracts: True, not this filter: False.

—is_signed (bool): Filter only contracts signed: True, not signed: False, both: None.

–with_event (bool): Filter only contracts with existing event: True, without existing event: False, both: None.

—is_paid (bool): Filter only contracts paid: True, not paid: False, both: None.
