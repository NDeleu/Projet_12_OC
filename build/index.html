Welcome to Project12’s documentation!¶

To access the application’s main menu, you can run python main.py or python3 main.py depending on your configuration.

The commands are entered in the header of each endpoint presented in this documentation.

Example: main.py collaborator create.

These are the commands you need to enter on the command line to access the query.

Nevertheless, please add the interpreter at the beginning of the command, here python, and according to your configuration.

Example: Creating a python collaborator main.py.

Or: creating a python3 collaborator main.py.
