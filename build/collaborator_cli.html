Collaborator¶

main.py collaborator create¶

Create a collaborator with the following arguments:

Parameters:

firstname: Firstname in alphabetical value.

lastname: Lastname in alphabetical value.

email: Email in alphabetical value and in form alpha@alpha.alpha.

role: Role in numerical value, administrator: 1, seller: 2, support: 3.

password: Password in form with letters, numbers, and a limited set of special characters (!@#$%^&*()_-).

main.py collaborator create [OPTIONS] FIRSTNAME LASTNAME EMAIL ROLE PASSWORD

main.py collaborator update¶

Update a collaborator with the following options:

Parameters:

collaborator_id: Collaborator ID in numerical value.

Options:

–firstname: For change: Firstname in alphabetical value, for keep unchanged: None.

—lastname: For change: Lastname in alphabetical value, for keep unchanged: None.

–email: For change: Email in alphabetical value and in form alpha@alpha.alpha, for keep unchanged: None.

—password: For change: Password in form with letters, numbers, and a limited set of special characters (!@#$%^&*()_-), for keep unchanged: None.

main.py collaborator update [OPTIONS] COLLABORATOR_ID
